# 📊 Bank Loan Dashboard (Power BI Project)

## 🔍 Project Overview

This project focuses on analyzing bank loan data using Power BI to gain meaningful insights into loan performance, customer behavior, and financial risk. The dashboard helps stakeholders make data-driven decisions by visualizing key loan metrics.

---

## 🎯 Objectives

* Analyze total loan applications and approvals
* Track funded amount and repayment status
* Identify high-risk customers and default trends
* Understand borrower demographics and patterns

---

## 📁 Dataset Details

The dataset includes:

* Loan Amount
* Interest Rate
* Loan Status (Approved / Rejected / Default)
* Customer Income & Employment Details
* Purpose of Loan

---

## 📊 Dashboard Features

* 📌 Total Loan Applications Overview
* 💰 Funded vs Received Amount Analysis
* 📉 Loan Status Distribution
* 📊 Interest Rate & Risk Analysis
* 👥 Customer Demographics (Income, Employment, etc.)

---

## 🛠️ Tools & Technologies Used

* Power BI (Dashboard & Visualization)
* Excel / CSV (Dataset)
* Data Cleaning & Transformation
